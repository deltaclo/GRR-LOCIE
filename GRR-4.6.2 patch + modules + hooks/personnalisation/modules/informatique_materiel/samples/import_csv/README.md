# Jeux CSV de test pour l'import Informatique materiel

Ces fichiers sont des extraits courts du classeur Excel historique.

Ordre conseille :

1. `02_personnes_excel_sample.csv`
2. `03_materiels_excel_sample.csv`
3. `04_prets_excel_sample.csv`
4. `05_prets_anomalies_sample.csv` pour tester le journal d'erreurs

Le fichier `01_categories_excel_sample.csv` est fourni pour tester la
previsualisation des categories ou une base sans categories. Sur une
installation normale du module, les categories historiques sont deja initialisees
automatiquement ; l'importer peut donc creer des doublons de categories.

Les fichiers utilisent le separateur `;` et des dates au format `AAAA-MM-JJ`.
